# 🛡️ Security Policy

## Supported Versions

| Version | Status     |
|---------|------------|
| 1.0.x   | ✅ Supported |

## Reporting Vulnerabilities

If you find a security issue or vulnerability, please contact us immediately:

- Email: **security@btcdigitalmedia.com**

Please do not open public issues for sensitive bugs.
